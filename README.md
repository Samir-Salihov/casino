## 🎨 Дизайн система

Проект построен на современной дизайн системе:

### 🎨 Цветовая палитра

| Тип | Темная тема | Светлая тема |
|-----|-------------|--------------|
| Фон | `#0b0f1a` | `#f6f8ff` |
| Поверхность | `#121829` | `#ffffff` |
| Акцент | `#2f7df6` | `#2f7df6` |
| Успех | `#29c07e` | `#1d985f` |
| Опасность | `#e05656` | `#c93f52` |
| Текст | `#e9f0ff` | `#0b0f1a` |

### ✨ Анимации

Все переходы сделаны с использованием функций `cubic-bezier` для натурального движения. Никаких резких скачков, каждая интеракция приятна глазу.

```css
animation: spin 1.2s cubic-bezier(.25,.6,.2,1) infinite
```


## 🚀 Как запустить

Проект не требует никакой сборки, зависимостей и серверов. Просто откройте в браузере:

```bash
# 1. Клонируйте репозиторий
git clone <url>

# 2. Перейдите в папку
cd casino

# 3. Откройте файл
open index.html
```

Или просто перетащите `index.html` в окно браузера. Всё готово к работе ✨
